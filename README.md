## Project Structure

프로젝트 폴더 구조

```
🔥 frontend🔥
├─config
│ └─webpack.config.js
├─public
│ ├─index.html (기본 HTML)
│ └─favicon.ico (파비콘)
├─src
│ ├─assets (이미지, 폰트 등)
│ ├─components (컴포넌트 like view)
│ ├─constants (상수, 라우트 경로)
│ ├─hooks (커스텀 훅)
│ ├─styles (글로벌 스타일)
│ ├─lib (리액트 라우터, styled-components)
│ ├─pages (페이지)
│ ├─saga (saga 함수)
│ ├─store (redux 모듈)
│ ├─styles (글로벌 스타일)
│ ├─utils (공통 유틸 함수)
│ │ └─api (api axios 요청)
│ └─index.js
├─.babelrc
├─.eslintrc.json
├─.prettierrc
├─.jsconfig.json
├─package.json
└─yarn.lock

🔥 backend🔥
├── build (프로젝트 빌드 결과)
└── src (소스코드)
    ├── main
    │   ├── java
    │   │   └── com
    │   │       └── infp
    │   │           └── ciat
    │   │               ├── board (게시판 패키지)
    │   │               ├── common (공통 패키지)
    │   │               ├── config (설정 패키지)
    │   │               ├── plant (식물 패키지)
    │   │               └── user (사용자 패키지)
    │   └── resources
    │       └── application.yml (스프링 프로파일)
    └── test (테스트 소스코드)
├── gradlew (gradlew 실행파일)
├── gradlew.bat (gradlew 실행파일)
├── settings.gradle (gradlew 설정파일)
└── build.gradle (gradlew 설정파일)
```
